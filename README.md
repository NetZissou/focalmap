## Installation

You can install the development version of opioidDashboard like so:
First, clone this repo to your server. Then, open the repo and type the
following command

``` r
library(devtools)
devtools::load_all()
devtools::install()
```

You can run the app in your local session by using the `app_dev()`
function.
